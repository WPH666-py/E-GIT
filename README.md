# e-git — 面向中文用户的本地 GitHub 管理面板

**e-git** 是一款便携式 GitHub 项目管理工具，基于 Python Flask 后端与 H5 前端构建，打包为独立 exe，无需安装任何依赖即可在任何 Windows 电脑上运行。

---

## 核心功能

### 1. 账号绑定
通过 GitHub Personal Access Token 绑定个人账号，支持 `repo`、`user`、`delete_repo` 等权限配置，绑定后即可使用全部功能。

### 2. 我的仓库
查看当前账号下所有 GitHub 仓库，支持一键刷新列表、创建新仓库、删除已有仓库。

### 3. 上传项目
支持拖拽文件和文件夹批量上传到指定仓库。底层递归遍历多层嵌套子目录，自动处理路径映射与文件去重，避免重复上传。

### 4. 我的分支
以可视化 Git 分支图谱（SVG 渲染）展示仓库所有分支的拓扑结构。每个提交节点可点击查看贡献者用户名、所在国家或地区、提交时间、修改文件清单及前后内容对比。支持创建分支、合并分支、删除分支操作，结果实时同步至 GitHub。

### 5. 我的版本
按仓库查询提交历史，支持**回滚**与**还原**功能。回滚将分支指针退回到历史提交节点，还原可撤销回滚恢复至回滚前状态，所有操作直接作用于 GitHub 远程仓库。

### 6. 下拉项目
从 Git 地址克隆项目到本地，支持 Clone 与 Download ZIP 两种方式。

### 7. 设置页面
包含作者联系方式、基础概念讲解（仓库、分支、节点、克隆、回滚、还原等）和绑定 GitHub 账号的图文教程。另提供代理与 VPN 配置指南，推荐 Clash Verge 代理工具及稳联云、飞鸟云、穿墙猫三个代理节点平台。

---

## 技术特点

- **零依赖运行**：PyInstaller 打包为单文件 exe，集成 Python 运行时与全部依赖
- **前后端一体**：Flask 提供 API 服务，纯 H5 前端无需额外 Web 服务器
- **GitHub API v3**：所有操作通过 GitHub REST API 实现，数据实时同步
- **可视化分支图谱**：模拟原版 Git 的 DAG 拓扑渲染，直观展示分支与提交关系
